# dancurr-CLI
Currency converting CLI

## Install

```bash
npm install dancurr -g
```

## Usage
```
usage: dancurr <int: ammount> <string: currency from> <string: currency to>

-h:  Show these instructions
```
## Example

```bash
  dancurr 100 usd isk
```
